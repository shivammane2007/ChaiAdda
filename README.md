# 🍵 ChaiAdda — The Soul of India, Brewed for the World.

A premium, immersive web experience built for an authentic Indian Chai café brand. Designed with meticulous attention to detail, this project combines traditional heritage with modern aesthetics.

![Theme Preview](/public/images/hero_bg.png)

## ✨ Core Features

- 🎭 **Cinematic Hero Experience**: Immersive, full-screen visuals featuring high-quality AI-generated assets.
- 📜 **Artisanal Storytelling**: A dedicated "Our Story" page narrating the brand's heritage from the gardens of Assam.
- 🍵 **Interactive Menu**: A category-filtered, visually rich menu showcasing premium chai blends like Saffron Gold and Masala Chai.
- 💼 **Franchise Hub**: A dedicated lead-generation landing page for business partners.
- 🌑 **Premium Aesthetics**: Harmonious dark-themed design with warm copper/tea-colored accents.
- 📱 **Fully Responsive**: Optimized for all devices, from mobile to desktop.

## 🛠️ Technology Stack

- **Core**: [Next.js 14+](https://nextjs.org) (App Router), React 19, TypeScript.
- **Styling**: [Vanilla CSS](https://developer.mozilla.org/en-US/docs/Web/CSS) for custom, high-performance designs.
- **Theming**: [next-themes](https://www.npmjs.com/package/next-themes) for seamless light/dark mode support.
- **Icons**: [Lucide React](https://lucide.dev).
- **Quality**: ESLint, TypeScript.

## 🚀 Getting Started

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/shivammane2007/ChaiAdda.git
    cd ChaiAdda
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

4.  **View locally**:
    Open [http://localhost:3000](http://localhost:3000) in your browser.
